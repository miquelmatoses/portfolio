## Design System (mm-design)

This site follows mm-design visual conventions (https://github.com/miquelmatoses/mm-design).

### Rules
1. **Fonts**: Playfair Display (headings), Roboto (body), JetBrains Mono (code). No Inter, no Fira Code.
2. **Badge colors**: All shields.io hex colors must use mm-design palette:
   - Red: cf3339 | Blue: 0047ba | Yellow: f1c22f | Green: 427c42 | Black: 111111
3. **CSS tokens**: Use --mm-* naming for any new tokens. Existing dark theme overrides are acceptable.
4. **Icons**: Use mm-design SVGs where applicable.
5. **No arbitrary hex colors**: Always use the closest mm-design brand color for accents.

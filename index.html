Open data · Demographics

poblacio-valenciana

125 years of historical population data for all 547 municipalities of the Valencian Community. Census records (1900-1991) + continuous register (1996-2025), consolidated and automated.

  • INE API integration — fetches from 6 official government statistics tables with rate limiting and error handling
  • Bilingual name normalization — 142-entry Valencian mapping, deduplication, and dissolved municipality filtering
  • GitHub Actions automation — quarterly pipeline fetches, cleans, generates bar chart race GIF, and auto-commits
  • CC0 public domain — 20,995 clean records ready for analysis, fully reproducible from source
